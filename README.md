# Simple Notes API

Built with Node.js, Express, and PostgreSQL.  
Implements authentication via JWT, raw SQL (no ORM), and full CRUD for personal notes.


##Tech Stack

- Node.js + Express
- PostgreSQL
- JWT for Auth
- bcrypt for password hashing
- Raw SQL via `pg` (no ORM)
- CORS enabled
